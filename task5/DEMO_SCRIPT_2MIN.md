# Demo Script (2 Minutes)

## 0:00 - 0:15 Intro

- This is Deadline Coach, a task-planning app for students.
- It helps users focus on urgent tasks and avoid missing deadlines.

## 0:15 - 0:40 Create tasks

- Set Active User (for example: andrey).
- Create 2-3 tasks with different deadlines.
- Show that tasks appear in All Tasks.

## 0:40 - 1:00 Today Plan logic

- Open Today Plan.
- Explain it now shows overdue and due-today tasks only.
- Mention this keeps the plan focused.

## 1:00 - 1:25 Version 2 feature - edit task

- Click Edit for one task.
- Change title/course/deadline/effort.
- Save changes and show updated result in list.

## 1:25 - 1:40 Complete task and filter

- Mark one task as done.
- Switch All Tasks filter between Open only and All tasks.

## 1:40 - 1:55 Multi-user isolation

- Change Active User to another ID.
- Show empty list or different data for that user.
- Explain each user sees only their own tasks.

## 1:55 - 2:00 Closing

- Summarize: Version 1 core flow + Version 2 editing and usability improvements.
